  • Terms and Equations



Terms and Equations

  • General


    • Phylogenetic Tree:

      A diagram representation of the evolutionary relationships between a group of organisms or biological sequences.


    • Bioinformatics:

      The process of collecting and analyzing complex biological data.


  • Multiple Sequence Alignment


    • Multiple Sequence Alignment:

      A sequence alignment of three or more biological sequences.


    • Point Mutation:

      A mutation which replaces a single nucleotide with another nucleotide.


  • Distance Matrix


    • Distance Matrix:

      A matrix which displays the pairwise distance between all sequences of interest.


    • Genetic Divergence:

      The process in which two sequences acquire independent genetic mutations over time.


    • Genetic Distance:

      A measure of the genetic divergence between sequences.


    • Pairwise Distance:

      $d = -\frac{3}{4} ln(1 - \frac{1}{4}p)$


  • UPGMA Method


    • Ultrametric Tree:

      A phylogenetic tree in which the distance from each leaf node to the root node is equal.


    • Clade:

      A group containing a single ancestor and all of its descendants.


    • Time Cost/Complexity:

      A functional representation of the time required to run an algorithm.


    • Distance Calculation:

      $d_{AB, X} = \frac{1}{2}(d_{AX} + d_{BX})$


  • Neighbor Joining Method


    • Operational Taxonomic Unit (OTU):

      A species represented by a biological sequence.


    • Additive Property:

      $d_{AB} = d{AX} + d_{BX}$.


    • Net Divergence:

      $r_{i} = \sum_{} d_{ik}$


    • Distance Calculation:

      $M_{ij} = d_{ij} - \frac{r_{i} - r_{j}}{N - 2} $


    • Branch Length Calculation:

      $S_{AX} = \frac{1}{2}d_{AB} + \frac{1}{2(N - 2)}(r_{A} - r_{B})$

      $S_{BX} = S_{AB} - S_{AX}$


  • Fitch-Margoliash Method


    • Distance From Branch Lengths:

      $d_{XY} = b_{1} + b_{2}$

      $d_{XZ} = b_{1} + b_{3}$

      $d_{YZ} = b_{2} + b_{3}$


    • Branch Lengths From Distances:

      $b_{1} = \frac{1}{2}(d_{XY} + d_{XZ} - d_{YZ})$

      $b_{2} = \frac{1}{2}(d_{XY} + d_{YZ} - d_{XZ})$

      $b_{3} = \frac{1}{2}(d_{XZ} + d_{YZ} - d_{XY})$


    • Distance Calculation:

      $d_{AB} = \frac{1}{N_{A}N_{B}}\sum\limits_{i \in A, j \in B}d_{ij}$


    • Distance Matrix Calculation:

      $d_{DA} = \frac{N_{B}d_{BA} + N_{C}d_{CA}}{N_{B} + N_{C}}$


  • Maximum Parsimony Method


    • Character:

      Any attribute along which a sequence may vary.


    • Character State:

      Any possible state of a character.


    • Informative Site:

      A position in an alignment which contains at least two different character states, with each character state occurring in at least two of the sequences.


  • Maximum Likelihood Method


    • Likelihood Calculation:

      $L = \sum_{j = 1}^{n} L_{j}$
