Distance Matrix


Now that we have a multiple sequence alignment, we need a way to represent the genetic distances between each sequence. This can be achieved through the construction of a distance matrix, which is a representation of the pairwise distances between a given sequence and each other sequence of interest, using the information provided by the alignment.


To calculate the pairwise distance, count the number of positions with mismatched nucleotides/amino acids between two sequences of interest. This number is divided by the length of the alignment to give a proportion of the number of mismatched sites between the sequences. For now, this proportion will be called $p$.

This application, when calculating distance matrices, will follow the Jukes-Cantor model for evolution, which holds that all point mutations in sequences are equally likely. To calculate a genetic distance that stays true to this model, the following equation must be used, where $d$ is the pairwise distance between two given sequences, and $p$ is defined above:


$d = -\frac{3}{4} ln(1 - \frac{1}{4}p)$




Distance matrices are used by distance-based tree-building methods which seek to group the sequences by either minimizing the genetic distance or maximizing the similarity between the sequences. However, character-based methods use different criteria when building trees. The strengths and weaknesses of each methods will be described in the following sections.
